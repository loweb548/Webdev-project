Chuck Norris System Karate

The Chuck Norris System (CNS), formerly known as Chun Kuk Do, is a martial arts system founded by the legendary Chuck Norris.

Core Elements of CNS

  1. Techniques
    • Spinning Back Kick
    • Roundhouse Kick
    • Knife Hand Strike
  2. Philosophy
    • Respect
    • Self-discipline
    • Indomitable spirit
  3. Training Components
    • Kata
    • Sparring
    • Self-defense drills
